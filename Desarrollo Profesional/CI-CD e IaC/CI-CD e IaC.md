# 🚀 CI/CD e Infraestructura como Código

[[Desarrollo Profesional/Inicio Profesional|⬅️ Volver a Desarrollo Profesional]]

> [!abstract] Sobre esta sección
> Tienes el código y tienes el clúster ([[Desarrollo Profesional/Kubernetes/Kubernetes|Kubernetes]]) — esta sección es el puente entre ambos. CI/CD automatiza el camino desde un commit hasta producción: compilar, testear, empaquetar y desplegar sin intervención manual y de forma repetible. La Infraestructura como Código (IaC) aplica la misma idea a la infraestructura: describes tu nube en ficheros versionados en vez de hacer clic en una consola. Y GitOps lleva el principio al extremo: el repositorio de git es la única fuente de verdad del estado deseado, y un agente reconcilia el clúster para que coincida.
