## 📅 Lista de Ideas / Planificación

- [ ] **1. Guía Paso a Paso: Implementando RAG desde Cero**
  - *Descripción:* Explicar cómo hacer una búsqueda vectorial simple utilizando Python, embeddings de OpenAI/Gemini y una base de datos vectorial como ChromaDB.
  - *Público objetivo:* Desarrolladores backend intermedios.
  - *Estado:* 📋 Idea.

- [ ] **2. Guía Práctica de Prompt Engineering en Producción**
  - *Descripción:* Recopilación de patrones de prompts como Chain of Thought, Few-shot prompting y estructuración de respuestas en formato JSON para APIs.
  - *Público objetivo:* Desarrolladores de aplicaciones de IA.
  - *Estado:* 📋 Idea.

- [ ] **3. LLMs Locales: Configuración y uso con Ollama y Python**
  - *Descripción:* Cómo correr modelos de código abierto (Llama 3, Mistral) localmente en tu propia máquina y conectarlos con tus scripts de desarrollo.
  - *Público objetivo:* DevOps y desarrolladores preocupados por la privacidad.
  - *Estado:* 📋 Idea.
