# ⚛️ React — Curso Completo

[[Desarrollo Profesional/Inicio Profesional|⬅️ Volver a Desarrollo Profesional]]

> [!abstract] Sobre este curso
> Curso de React moderno de principiante a avanzado, con **TypeScript (TSX)**, **Vite** y **hooks** (React 18/19). Cada página es autocontenida y se enlaza con las anteriores. El enfoque es el React actual: componentes de función, hooks y composición — nada de clases legacy salvo para contextualizar. Sirve como ruta de aprendizaje secuencial o como referencia rápida.
