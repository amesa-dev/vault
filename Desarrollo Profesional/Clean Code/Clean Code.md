# 🧹 Clean Code

[[Desarrollo Profesional/Inicio Profesional|⬅️ Volver a Desarrollo Profesional]]

> [!abstract] Sobre esta sección
> Clean Code no es una colección de reglas de estilo — es un conjunto de principios para escribir código que comunica su intención claramente, que otros pueden modificar sin miedo y que el tiempo no pudre. Robert C. Martin ("Uncle Bob") sistematizó estos principios, aunque muchos vienen de la tradición del software profesional desde los años 70. Los ejemplos en Python.
